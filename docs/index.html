This section will explain how to set up the Entra Security Monitor app.

Step 1: Azure App Registration

  • Go to Azure Active Directory â†’ App registrations
  • Register a new application named Entra Monitor
  • Set the platform to Web and use the redirect URI: https://YOUR-SITE.azurewebsites.net

Step 2: API Permissions

Add the following Microsoft Graph permissions (Application type):

  • Directory.Read.All
  • AuditLog.Read.All
  • Policy.Read.All
  • App.Read.All

Grant Admin Consent.

Step 3: Deploy the Platform

Click the Deploy to Azure button from the homepage to automatically provision the Web App and deploy the code.
