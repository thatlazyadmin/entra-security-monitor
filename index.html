  1. Deploy the Web App using the button above.
  2. Create an App Registration in Entra ID.
  3. Assign Application.Read.All, AuditLog.Read.All, Policy.Read.All permissions.
  4. Generate a client secret (minimum 6 months validity).
  5. Paste the App ID and Secret into Azure Web App configuration settings.
  6. Restart the Web App.
